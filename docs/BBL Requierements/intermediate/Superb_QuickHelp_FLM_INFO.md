# Flächenmanagement Info - Startbildschirm

Der Startbildschirm von Flächenmanagement Info zeigt Ihnen eine geografische Übersichtkarte aller verfügbaren Information in Gebäuden und Grundstücken des zivilen Immobilienportfolio BBL zum Tagesdatum an. 

Die Standorte, an denen sich die Gebäude befinden, werden durch zwei Arten von Symbolen dargestellt:

- Ein Spot                        steht für ein einzelnes Gebäude, dessen Name und Kürzel oberhalb des Spots angegeben sind.
- Ein Cluster                 steht für zwei oder mehr Gebäude, deren Anzahl mit einer Ziffer oben rechts angegeben wird.

# Flächenmanagement Info - Suchmethode

In der Menüleiste im Bereich Suche sind folgende Such-/Eingabefelder zu sehen.

Zur Verfügung stehen Ihnen die Auswahl-Felder:

- Fernglas – Ein-/Ausblenden der Suchergebnisse
- Datum – Das Datum ist der Stichtag der Auswahl für die Suchelemente
- Suchmethoden – Auswahl der verfügbaren Suchmethoden (bspw. Gebäude/Grundstück nach AOID)
- Suchbegriff – Eingabe eines Suchbegriffes zur Einschränkung der Suchergebnisse (bspw. Fellerstrasse)

# Flächenmanagement Info – Objektnavigator

Als Objekt-Navigator wird jene Gruppe von Feldern in der Navigationsleiste bezeichnet, mit Hilfe derer Sie die ausgewählten Gebäude und deren Ebenen aufrufen können:

	Objekt-Navigator

Die mit dem Objekt-Navigator abgebildete Objekthierarchie wird automatisch beim Klick auf einen Gebäude-Spot in der Karte, bei der Auswahl eines Gebäudes über den Verlauf oder durch Auswahl eines Suchergebnisse befüllt. Gibt es zu der aktuellen Objektauswahl weitere untergeordnete Objekte, wird automatisch die Liste der untergeordneten Objekte in Form eines Auswahlmenüs angezeigt.

Je nach Objekt halten die jeweiligen Menüeinträge verschiedene Informationen parat:

Es kann vorkommen, dass auf der linken Seite eines Menüeintrags gar kein Icon eingeblendet wird. Das bedeutet, dass für dieses Objekt keine Plan-Grafik zur Verfügung steht oder der Stichtag vor 01.01.2024 eingegeben wurde.

Es besteht die Möglichkeit, dass auf der linken Seite statt der Büroklammer ein Schloss zu sehen ist. In diesem Fall fehlt Ihnen die notwendige Zugriffsberechtigung zu diesem Objekt.

# Flächenmanagement Info – Raumliste ohne grafischen Pläne

# Flächenmanagement Info – Tipps

Mit Stichtag bestimmen Sie den Stichtag für Ihre Suche. Default ist immer der aktuelle Tag. Für Ihre Suche ist es wichtig, das korrekte Datum zu setzen, da dieses die Gültigkeit von Wirtschaftseinheiten, Gebäude, Grundstücke und Räumen steuert und Sie nur dann die gesuchten Inhalte angezeigt bekommen, wenn Sie den korrekten Stichtag einstellen.

Ein Gebäude oder Grundstück wird erst dann neu in den Verlauf aufgenommen, wenn Sie eine Ebene darin öffnen. Hierdurch wird vermieden, dass versehentliche Klicks den Verlauf unnötig füllen.

Durch eine tabellarische Auswertung wird die Ergebnisanzeige einer zuvor vorgenommen grafischen Auswertung nicht beeinflusst. Damit kann eine tabellarische Auswertung mit einer beliebigen grafischen Auswertung kombiniert werden. Ist das nicht gewünscht, muss vor der tabellarischen Auswertung die vorherige grafische Auswertung über das Icon                      entfernt werden.

Der Report DIN277 zeigt die Quadratmeter der jeweiligen Flächen und die zugeordnete Zahl an Räumen an. Durch Anklicken der Anzahl Räume werden diese im Plan eingefärbt.
